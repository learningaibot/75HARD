Questions

  
FAQ

  

    
How do I add a buddy?+
Go to the Buddy System section and type their exact username. They'll receive a friend request and can accept or decline. Once accepted, you'll see each other's progress in real-time.

    
Are my progress photos private?+
You choose per photo: "Private" stores it only in your browser. "Visible to Buddies" uploads to Supabase Storage so accepted buddies can see it in the live feed.

    
How do I customise the tasks?+
Click the Admin button in the nav bar and go to the Tasks tab. You can rename, reorder, add, or remove tasks — for example changing them to "Read Quran" or "Pray Namaz".

    
Can I change the challenge length?+
Yes — Admin Panel → Challenge tab → set your total days. Useful for Ramadan 30-day challenges, 30 Clean, or any custom duration.

    
How do I host this on GitHub Pages?+
1. Create a GitHub repo and push this index.html file. 2. Settings → Pages → Source: Deploy from branch → main. 3. Your site is live at username.github.io/repo-name. Everyone uses the same file but their own Supabase credentials.

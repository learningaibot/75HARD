Common Questions

FAQ

What happens if I miss one task?+
You start over from Day 1 — no exceptions. Missing even one task on any single day means a full restart. That's the entire point.
Can the admin customize the tasks?+
Yes! Click the Admin button in the nav to open the admin panel. You can add, remove, and rename tasks, set the challenge duration, configure selfie privacy defaults, and manage buddies. The default admin password is "admin123".
Are my progress photos private?+
Photos are stored locally in your browser only — they never upload to any server. You can mark each photo as Private (only you) or Shared (visible in the buddy feed if you choose to share). The admin can set the default privacy mode.
Can I change the challenge length?+
Yes — in the Admin Panel, go to Challenge Settings and set your desired number of days (1–365). The calendar and stats update automatically.
How do I add custom tasks like "Read Quran" or "Pray Namaz"?+
Open the Admin Panel → Tasks tab. You can rename any existing task, edit its description, add new tasks, or remove tasks you don't need. Changes apply immediately to the tracker.
